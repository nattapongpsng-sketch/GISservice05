Frontend สำหรับทดสอบ ArcGIS REST → Leaflet → Network Tracking → Fault Current API → Fuse Coordination

## Structure

```text
PEA-GIS-Protection/
├── index.html
├── css/

├── js/
│   ├── config.js
│   ├── map.js

│   ├── tracking.js
│   ├── fault.js
│   ├── api.js
```

## Usage

1. เปิด `index.html`
2. กดโหลดข้อมูล
3. คลิก Switch/Fuse/DF
4. กด Track + คำนวณ Fault
